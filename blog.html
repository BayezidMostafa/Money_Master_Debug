QUESTION #1
What is the difference between local storage and session storage?

Local Storage

Session Storage

If we set any kind of data in local storage, that will not go anywhere even after a reload or closing of the tab.
The storage capacity of local storage is 5/10MB.
We only can delete the local storage data using Javascript or Manually. Because all the existing data will remain after reloading or closing the tab.
The client can read the local storage data.

If we set data in session storage, that will be gone after a reload or closing of the tab.
The storage capacity of session storage is 5MB.
We don't have to delete session storage data. Because Those existing data will disappear automatically after a reload or closing of the tab.
The client can not read session storage data.

QUESTION #2
What is the difference between Global scope and Block scope?

Global Scope

Block Scope

The Global scope is a scope that is visible from all other scopes. A variable that is declared outside of a function is considered as a Global scope variable. This variable can be accessed from anywhere in a javascript program.

The scope is created with a pair of curly braces called block scope. It could be in a function, condition, or loop. When a variable declares in it, it will be considered as a block scope variable. We can access the block scope variable inside the block only

QUESTION #3
How does the event loop work in JavaScript?

The event loop is a continuously running process that monitors the callback queue and the call stack as well. First of all, It will check the call stack. If the call stack is empty then it will call the first event from the queue and push it to the call stack. If the call-back queue is empty, nothing will gonna happen.

QUESTION #4
The ways we can get undefined

8 ways we can get undefined:
1. If we called a variable, which is not initialized yet.
2. If a function doesn't have any return.
3. Parameter that is not passed.
4. If there is no statement on the right side of the return.
5. Try to get a property that doesn't exist in an object.
6. Try to access an array element that is outside of the index range.
7. Delete an array element using delete.
8. Set a value directly to undefined.
